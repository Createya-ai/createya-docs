# 3.1

> 3.1

**Стоимость:** 192 кредитов / запрос (duration: 4s=96кр, 6s=144кр, 7s=168кр, 8s=192кр; resolution: 720p=192кр, 1080p=192кр, 4k=384кр).

**Family slug:** `veo` · **Output:** video · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo-3.1-t2v",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "veo-3.1-t2v", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "veo-3.1-t2v",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `duration` | enum |  | Длительность видео в секундах |
| `aspect_ratio` | enum |  | Соотношение сторон сгенерированного изображения/видео |
| `generate_audio` | boolean |  | Генерировать аудио для видео |
| `auto_fix` | boolean |  | Автоматически исправлять промпты, не прошедшие проверку контента |
| `resolution` | enum |  | Разрешение генерации |
| `seed` | integer |  | Сид для генератора случайных чисел. Одинаковый сид даёт одинаковый результат. |
| `negative_prompt` | string |  | Описание нежелательных элементов в генерации |
