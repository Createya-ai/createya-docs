# 3.1 Fast

> 3.1 Fast

**Стоимость:** 96 кредитов / запрос (resolution: 720p=96кр, 1080p=96кр, 4k=288кр; duration: 4s=48кр, 6s=72кр, 7s=84кр, 8s=96кр).

**Family slug:** `veo` · **Output:** video · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo-3.1-fast-t2v",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "veo-3.1-fast-t2v", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "veo-3.1-fast-t2v",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `generate_audio` | boolean |  | Нужно ли генерировать аудио для видео. |
| `duration` | enum |  | Длительность генерируемого видео. |
| `resolution` | enum |  | Разрешение генерируемого видео. |
| `aspect_ratio` | enum |  | Соотношение сторон генерируемого видео |
| `negative_prompt` | string |  | Отрицательный промпт для управления генерацией видео. |
| `safety_tolerance` | enum |  | Уровень строгости модерации контента. 1 — самый строгий (блокирует большую часть контента), 6 — наименее строгий. |
| `seed` | integer |  | Сид (seed) для генератора случайных чисел. |
| `auto_fix` | boolean |  | Нужно ли автоматически пытаться исправлять промпты, не прошедшие проверку политики контента или другие проверки, путем их переписывания. |
