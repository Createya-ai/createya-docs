# 2

> 2

**Стоимость:** 48 кредитов / запрос (12кр/сек · 4–12 сек).

**Family slug:** `sora` · **Output:** video · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "sora-2-i2v",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "sora-2-i2v", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "sora-2-i2v",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `aspect_ratio` | enum |  | Соотношение сторон сгенерированного видео |
| `resolution` | enum |  | Разрешение сгенерированного видео |
| `duration` | integer |  | Длительность сгенерированного видео в секундах |
| `model` | enum |  | Модель для генерации. При выборе модели по умолчанию будет использоваться последняя версия (snapshot), в противном случае выберите конкретную версию модели. |
| `delete_video` | boolean |  | Удалять ли видео после генерации в целях конфиденциальности. Если выбрано значение True, видео нельзя будет использовать для ремиксов, и оно будет безвозвратно удалено. |
| `detect_and_block_ip` | boolean |  | Если этот параметр включен, промпт (и изображение для режима image-to-video) будет проверяться на наличие ссылок на известную интеллектуальную собственность, и запрос будет заблокирован при их обнаружении. |
