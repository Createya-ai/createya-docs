# v3.0

> v3.0

**Стоимость:** 5 кредитов / запрос (style: 5–10кр (85 вариантов)).

**Family slug:** `recraft` · **Output:** image · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "recraft-v3-i2i",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "recraft-v3-i2i", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "recraft-v3-i2i",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `style` | enum |  | Стиль генерируемых изображений. Векторные изображения стоят в 2 раза дороже. |
| `strength` | number |  | Определяет отличие от оригинала, значение от 0 до 1, где 0 — почти идентично, а 1 — минимальное сходство |
| `colors` | repeater |  | Массив предпочтительных цветов |
| `negative_prompt` | string |  | Текстовое описание нежелательных элементов на изображении |
