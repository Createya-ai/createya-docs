# 1.5

> 1.5

**Стоимость:** 3 кредитов / запрос (× num_images (1–4)).

**Family slug:** `grok` · **Output:** image · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "grok-imagine-i2i",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "grok-imagine-i2i", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "grok-imagine-i2i",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `num_images` | integer |  | Количество изображений для генерации |
| `output_format` | enum |  | Формат сгенерированного изображения |
