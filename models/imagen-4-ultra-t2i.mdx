# v4

> v4

**Стоимость:** 8 кредитов / запрос (× num_images (1–4)).

**Family slug:** `imagen` · **Output:** image · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "imagen-4-ultra-t2i",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "imagen-4-ultra-t2i", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "imagen-4-ultra-t2i",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `num_images` | integer |  | Количество генерируемых изображений. |
| `resolution` | enum |  | Разрешение генерируемого изображения. |
| `aspect_ratio` | enum |  | Соотношение сторон генерируемого изображения. |
| `output_format` | enum |  | Формат генерируемого изображения. |
| `safety_tolerance` | enum |  | Уровень модерации контента. 1 — самый строгий (блокирует большую часть контента), 6 — наименее строгий. |
| `seed` | integer |  | Сид (seed) для генератора случайных чисел. |
