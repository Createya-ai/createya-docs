# V3 pro

> V3 pro

**Стоимость:** 70 кредитов / запрос (14кр/сек · 3–15 сек).

**Family slug:** `kling` · **Output:** video · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-video-v3-pro-i2v",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "kling-video-v3-pro-i2v", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "kling-video-v3-pro-i2v",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `duration` | enum |  | Длительность генерируемого видео в секундах |
| `voice_ids` | array |  | Дополнительные ID голосов для генерации видео. Ссылайтесь на голоса в промпте с помощью <<<voice_1>>> и <<<voice_2>>> (максимум 2 голоса на задачу). Получить ID голосов можно через эндпоинт kling video create-voice: https://createya/models/createya/kling-video/create-voice |
| `multi_prompt` | array |  | Список промптов для генерации многокадрового видео. Если указано, разделяет видео на несколько сцен. |
| `generate_audio` | boolean |  | Нужно ли генерировать нативное аудио для видео. Поддерживается озвучка на китайском и английском языках. Другие языки автоматически переводятся на английский. Для английской речи используйте строчные буквы; для аббревиатур или имен собственных — заглавные. |
| `shot_type` | string |  | Тип генерации многокадрового видео. Требуется, если указан multi_prompt. |
| `negative_prompt` | string |  | Негативный промпт |
| `cfg_scale` | number |  | Ниже — больше свободы и естественности. Выше — точнее следование промпту, но результат может стать менее живым. |
