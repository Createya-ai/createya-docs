**Family slug:** `seedance` · **Output:** video · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0-i2v",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "seedance-2.0-i2v", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "seedance-2.0-i2v",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `resolution` | enum |  | Разрешение видео: 480p для быстрой генерации, 720p для баланса. |
| `duration` | enum |  | Длительность видео в секундах. Поддерживается от 4 до 15 секунд или «Авто», чтобы модель определила длительность на основе промпта. |
| `aspect_ratio` | enum |  | Соотношение сторон генерируемого видео. Используйте 16:9 для горизонтального формата, 9:16 для вертикального, 1:1 для квадратного, 21:9 для ультраширокого киноформата или «Авто» для определения по входному изображению. |
| `generate_audio` | boolean |  | Нужно ли генерировать синхронизированное аудио для видео, включая звуковые эффекты, фоновые звуки и синхронизацию губ (lip-sync). Стоимость генерации видео одинакова независимо от наличия аудио. |
| `seed` | integer |  | Случайное число (seed) для воспроизводимости. Обратите внимание, что результаты могут незначительно отличаться даже при одинаковом значении seed. |
| `end_user_id` | string |  | Уникальный идентификатор конечного пользователя. |
