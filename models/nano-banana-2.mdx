# Text to Image

> Faster. Sharper. Smarter.

**Стоимость:** 10 кредитов / запрос (× num_images; resolution: 1K→1x, 2K→1.5x, 4K→2x; +0.015 USD if enable_web_search=true).

**Family slug:** `nano` · **Output:** image · **Mode:** sync

📚 [Полный обзор модели на createya.ai](https://createya.ai/knowledge/nano-banana-2)

## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana-2",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "nano-banana-2", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "nano-banana-2",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `num_images` | integer |  | Количество генерируемых изображений. |
| `enable_web_search` | boolean |  | Включить веб-поиск для задачи генерации изображений. Это позволит модели использовать актуальную информацию из интернета для создания изображения. |
| `resolution` | enum |  | Разрешение генерируемого изображения. |
| `aspect_ratio` | enum |  | Соотношение сторон генерируемого изображения. Поддерживаются экстремальные значения: 4:1, 1:4, 8:1, 1:8. Используйте «auto», чтобы модель выбрала соотношение на основе промпта. |
| `output_format` | enum |  | Формат генерируемого изображения. |
| `thinking_level` | enum |  | Если параметр задан, включает размышление модели выбранного уровня («minimal» или «high») и добавляет процесс мышления в генерацию. Оставьте пустым, чтобы отключить. |
| `safety_tolerance` | enum |  | Уровень строгости модерации контента. 1 — самый строгий (блокирует большую часть контента), 6 — наименее строгий. |
| `seed` | integer |  | Сид (seed) для генератора случайных чисел. |
