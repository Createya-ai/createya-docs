# O3 pro

> O3 pro

**Стоимость:** 70 кредитов / запрос (14кр/сек · 3–15 сек).

**Family slug:** `kling` · **Output:** video · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-video-o3-pro-t2v",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "kling-video-o3-pro-t2v", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "kling-video-o3-pro-t2v",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `duration` | enum |  | Длительность видео в секундах (3-15с). |
| `aspect_ratio` | enum |  | Соотношение сторон генерируемого видео. |
| `generate_audio` | boolean |  | Нужно ли генерировать оригинальное аудио для видео. |
| `voice_ids` | array |  | Дополнительные ID голосов для генерации видео. Ссылайтесь на голоса в промпте с помощью <<<voice_1>>> и <<<voice_2>>> (максимум 2 голоса на задачу). Получить ID голосов можно через эндпоинт kling video create-voice: https://createya/models/createya/kling-video/create-voice |
| `multi_prompt` | array |  | Список промптов для многокадровой генерации видео. |
| `shot_type` | string |  | Тип многокадровой генерации видео. |
