# v1.0

> v1.0

**Стоимость:** 12 кредитов / запрос (12 кредитов за запрос).

**Family slug:** `midjourney` · **Output:** image · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "midjourney-t2i",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "midjourney-t2i", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "midjourney-t2i",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `aspect_ratio` | enum |  | Выбирает формат кадра: квадратный, горизонтальный или вертикальный. |
| `quality` | enum |  | Влияет на баланс между скоростью генерации и проработкой деталей. Меньшие значения работают быстрее, большие — дают более детализированный результат. |
| `stylize` | integer |  | Определяет, насколько сильно фирменный художественный стиль Midjourney влияет на результат: ниже — ближе к промту, выше — красивее и “по-midjourney”, но с меньшей точностью к мелким деталям запроса. |
| `chaos` | integer |  | Управляет вариативностью и непредсказуемостью результата. 0 — максимально спокойно, большие значения дают более разнообразные результаты. |
| `weird` | integer |  | Добавляет сюрреализм и экспериментальность. 0 — без эффекта, очень высокие значения дают экстремально необычные результаты. |
| `version` | enum |  | Позволяет выбрать версию Midjourney. |
| `niji` | enum |  | Использует аниме-ориентированный рендер вместо стандартного движка Midjourney |
| `seed` | integer |  | Use the seed parameter for testing and experimentation. Use the same seed and prompt to get similar results. |
