**Стоимость:** 4 кредитов / запрос (× num_images (1–4); +2кр с output_extra_megapixels).

**Family slug:** `flux` · **Output:** image · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "flux-2-pro-t2i",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "flux-2-pro-t2i", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "flux-2-pro-t2i",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `image_size` | enum |  | Определяет формат изображения: квадрат, вертикальный или горизонтальный. |
| `width` | integer |  | Ширина изображения в пикселях |
| `height` | integer |  | Высота изображения в пикселях |
| `output_format` | enum |  | Формат файла результата. |
| `safety_tolerance` | enum |  | Уровень допустимой безопасности для генерируемого изображения. 1 — самый строгий, 5 — самый разрешающий. |
| `enable_safety_checker` | boolean |  | Включить ли проверку безопасности. |
| `seed` | integer |  | Сид (seed), используемый для генерации. |
