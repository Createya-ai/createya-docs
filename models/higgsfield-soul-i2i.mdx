# v1

> v1

**Стоимость:** 12 кредитов / запрос (quality: medium=12кр, high=24кр).

**Family slug:** `higgsfield` · **Output:** image · **Mode:** sync



## Быстрый старт

### curl

```bash
curl -X POST https://api.createya.ai/v1/run \
  -H "Authorization: Bearer $CREATEYA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "higgsfield-soul-i2i",
    "input": {
      "prompt": "your prompt"
    }
  }'
```

### Python

```python
import os, requests
KEY = os.environ["CREATEYA_API_KEY"]
r = requests.post(
    "https://api.createya.ai/v1/run",
    headers={"Authorization": f"Bearer {KEY}"},
    json={"model": "higgsfield-soul-i2i", "input": {"prompt": "your prompt"}}
)
print(r.json())
```

### Node.js

```javascript
const r = await fetch("https://api.createya.ai/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CREATEYA_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "higgsfield-soul-i2i",
    input: { prompt: "your prompt" }
  })
});
console.log(await r.json());
```

## Параметры

| Параметр | Тип | Required | Описание |
|---|---|---|---|
| `size` | enum |  | Выбирает итоговый размер изображения в пикселях |
| `style` | enum |  | Готовый визуальный пресет для обработки изображения: от реалистичных и editorial-образов до более атмосферных и эффектных вариантов. |
| `strength` | number |  | Определяет, насколько сильно модель будет отходить от исходного изображения. |
| `quality` | enum |  | Выбирает качество результата: medium или high |
| `seed` | integer |  | The random seed to use for the generation. -1 means a random seed will be used. |
